# Security Policy

## Reporting a Vulnerability
If you discover a security vulnerability in any IDeeps project, please do NOT open a public issue.

Instead, report it privately by emailing: **security@ideeps.ai** (or info@ideeps.ai if the former is unavailable).

Please include:
- A description of the vulnerability
- Steps to reproduce
- Potential impact
- Any suggested mitigation (optional)

## Response Timeline
- We will acknowledge your report within 72 hours.
- We will provide a more detailed response within 7 days, including our assessment and next steps.
- We will keep you informed about the progress of the fix.

## Disclosure Policy
We follow coordinated disclosure. Please allow us a reasonable time to investigate and release a fix before disclosing the issue publicly.

Thank you for helping keep IDeeps and our users safe.
